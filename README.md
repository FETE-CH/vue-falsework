# vue-falsework

## Project setup
```bash
yarn
```

### Compiles and hot-reloads for development
```bash
yarn start
```

### Compiles and minifies for production
```bash
yarn build
```

### Lints and fixes files
```bash
yarn lint
```

### Run on Docker
```bash
docker build -t vue-falsework:0.0.1 . && docker run -p 80:80 --name vue-demo vue-test:0.0.1 
```
